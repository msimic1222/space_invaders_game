This project is a browser-based implementation of the classic Space Invaders arcade game built using JavaScript and HTML. The game recreates the core mechanics of the original arcade game where the player controls a spaceship and must defeat waves of alien invaders before they reach the bottom of the screen.

The player can move the spaceship left or right and fire laser beams upward to destroy incoming aliens. As aliens are eliminated, their movement speed gradually increases, making the game progressively more challenging. The game also includes visual explosion effects, a score counter, player lives, and clear win/lose game states.

The project was developed as part of a university assignment to demonstrate understanding of JavaScript programming, event handling, animation loops, collision detection, and browser-based game development.
